# Heartbeat

 A CakePHP plugin providing an application heartbeat status page with configurable sensors.

## Compatibility

| Plugin Version | CakePHP Compatibility | Branch      | Status    |
| -------------- | --------------------- | ----------- | --------- |
| 2.x            | 3.6+                  | cakephp-3.x | Supported |
| 1.x            | 2.x                   | cakephp-2.x | Supported |
| 0.x            | 2.x                   | -           | EOL       |
